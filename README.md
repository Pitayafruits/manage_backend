# 物业后台管理系统-后端

### 介绍
这是一款前后端分离的基于Spring Boot的物业管理系统，该系统实现了物业工作人员管理、业主管理、房屋管理、物业管理、人事管理、报修管理等主要功能。

### 软件架构
* 核心框架：Spring Boot 2.3.7.RELEASE
* 安全框架：Spring Security 5.3.6.RELEASE
* 持久层框架：MyBatis-Plus 3.5.2
* 关系型数据库: Mysql 5.7.35
* 数据库连接池：Druid 1.2.11
* 缓存数据库: Redis 4.0.9
* 项目管理工具: Maven 3.8.4
